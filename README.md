# DAY 1: PasteCV (AI)

## Problem
Recruiters receive CVs in inconsistent formats and waste time extracting key info manually.

## Why it matters
A recruiter reviewing 50 CVs/day loses 2–3 hours to formatting noise.

## Core feature
Paste raw CV text → structured JSON output (name, skills, experience, education).

## Tech stack
Flask · Anthropic Claude · Docker · React

## What ships
Web app with textarea input + structured JSON output display + copy button.

**repo**: pastecv  
**live**: Web app  
**infra**: Docker multi-stage build
